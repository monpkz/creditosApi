## CreditosApi
Ejemplo pequeño de API Rest de un sistema de creditos que nos permite registrar una lista de clientes y crearle un préstamo.


:toolbox:	 Creada 
  * JAVA11
  * SpringBoot
  * Hibernate
  
🛢  Base de Datos
  * MySql

  * Postgres.
